Tipos de Etiquetas en HTML

  • Etiqueta del tipo Block (en bloque)
  • Etiqueta del tipo INLINE (en linea)
  • Etiqueta del tipo "inline-block"

Tipo Block (en bloque)

Utiliza todo el espacio disponible del viewport (ancho), tambien permite modificar la proporcion ancho y alto del elemento. ejemplos:
- div
- titulos (h1..h6)
- etiquetas semanticas
- parrafos

Tipo Inline (en linea)

Solo utiliza el espacio necesario y no permite controlar la proporcion (height y width) ejemplos:
- button
- Etiquetas para aplicar texto enriquecido (span, i, b, mark, del, strong)
- a (enlacees)
- imagenes (img)
